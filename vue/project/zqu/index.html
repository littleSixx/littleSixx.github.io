    # 封装组件时

    • 封装组件时，组件的样式最好在组件内调整，否则会增加组件复用的难度
    • 如果是涉及到组件整体位置的样式，那就交给父组件来调整吧。

    # 关于position: fixed踩坑

    (32条消息) position: fixed如何相对父元素定位_天猫精灵998的博客-CSDN博客_css fixed 相对父元素 (opens new window)

    • 固定定位的元素实际上会受到父元素的影响，导致“固定定位不固定”
      • 如果父元素设置了transform: translate，子元素设置fixed定位的话产生不固定的现象！！！！
    • 虽然position:fixed是对于浏览器窗口定位的，但是只要不设置fixed元素的top，bottom，left，right，此时该元素就是相对于父元素进行定位的。如果要移动元素的位置，可以通过margin来实现。

    # 如何让一个元素相对于父元素固定定位

    应用场景：回退按钮、返回顶部按钮、zqu项目的导航栏。

    解决方法：(还没试过，不知道行不行)父级元素设置position:relative/absolute，子元素position:fixed，然后使用margin进行上下左右偏移就可以了。若使用了left或者right，top或者bottom，那么这4个属性会相对于窗口进行计算。
