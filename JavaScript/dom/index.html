    # 什么是DOM

    • DOM，全称Document Object Model 文档 对象 模型
      • 文档：表示的是整个html网页文档
      • 对象：表示将网页中的每一个部分(结点)都转换为了一个对象
      • 模型：使用模型来表示对象之间的关系，这样方便获取对象。体现结点之间的关系(兄弟、父子啥的)
    • js通过DOM来对html文档进行操作

    # 什么是节点

    常用的节点分为四类：

    1. 文档节点：整个html文档,nodeType为9
    2. 元素节点：html文档中的html标签,nodeType为1
    3. 属性节点：元素的属性,nodeType为2
    4. 文本节点：html标签中的文本内容,nodeType为3

    例：

    <p id="pId">This is a paragraph</p>
<-------------元素节点------------->
  <-属性节点->
            <-------文本节点---->

    # 获取元素

    • let ele = document.querySelector('')可以使用css选择器来选中元素。IE8及以上支持。**==注意：==**使用此方式只能选中一个，不能同时选多个。若存在多个，则默认选中第一个

    • let ele = document.querySelectorAll('')。此方式会返回数组，即使符合条件的只有一个元素，也会返回数组

    • let btn = getElementById("btn")

    • let btn = getElementsByName("btn")，通过name属性获取。可以同时获取多个元素，该方法会返回一个类数组对象，所有查询到的元素都会封装到对象中。

    • let btn = getElementsByTagName("btn")，同上

    • let btn = getElementsByClassName("btn")，同上，注意不是Class而是ClassName。IE9及以上才支持
